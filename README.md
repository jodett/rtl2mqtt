## Environment Variables

| Name      | Component | Description |
| --------- | --------- | ----------- |
| MQTT_HOST | RabbitMq  | Hostname    |
| MQTT_USER | RabbitMq  | Username    |
| MQTT_PASS | RabbitMq  | Password    |
| MQTT_PORT | RabbitMq  | Port (5672) |
| TZ        | System    | Timezone    |
